This Currency Converter app allows users to quickly and easily convert money between different currencies such as USD, EUR, CAD, AUD, JPY and GBP.

The app uses real-time exchange rates from an external API to ensure accurate and up-to-date conversions. Users can select their desired currencies, enter an amount, and instantly see the converted value.

This project was built as part of a web development course to demonstrate skills in HTML, CSS, JavaScript, and working with APIs.

The goal of this app is to provide a simple, fast, and user-friendly interface for everyday currency conversion needs.
