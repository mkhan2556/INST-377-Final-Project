# Currency Converter App

# Project Overview
The Currency Converter Web Application is a full-stack web application that allows users to convert monetary values between different currencies in real time. The system uses a Node.js backend (deployed as serverless functions on Vercel) and a frontend built with HTML, CSS, and JavaScript.

Users can input an amount, select currencies, and receive accurate conversion results along with exchange rates and timestamps. The application also stores conversion history in a Supabase database and visualizes the data using charts.

# Purpose
This project demonstrates:
- Full-stack web development using Node.js
- Integration with external APIs
- Database interaction using Supabase
- Client-server communication using Fetch API
- Data visualization using JavaScript libraries
- Deployment using Vercel

The application is designed to be simple, user-friendly, and scalable for future improvements.
